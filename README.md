# MAIN FEATURES
- 위치 기반 서비스를 통한 장보기 메이트 구하기 
- 게시글 목록에서 원하는 식재료, 마트, 지역 별로 확인하고 참여하기 
- 거래를 희망하는 유저와 채팅으로 소통하기
- 관심 가는 공구 포스팅 목록을 따로 관리하기 
- 알람 기능을 통해 거래 시간에 맞춰 공구 및 나눔에 참여
- 사용자간 거래 평가 및 레벨 확인
- 또 만나고 싶은 장보기 메이트 모아보기
